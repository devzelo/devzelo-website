# Devzelo

Future-Proof Software Solutions — static site built with React, Tailwind CSS, Framer Motion.

## Deploy on Vercel

1. Push this repository to GitHub.
2. Go to [vercel.com](https://vercel.com) → **Add New Project** → import the repo.
3. Framework Preset: **Other** (no build step needed — this is a pre-built static site).
4. Leave Build Command and Output Directory **blank**.
5. Click **Deploy**.

## Local Development

Serve the project with any static file server, for example:

```bash
npx serve .
```

Then open http://localhost:3000.
